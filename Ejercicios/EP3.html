Ejercicios propuestos tercer examen parcial

Ejercicio

Sea $\{A_n : n \in \mathbb{N}\}$ una sucesión de eventos. Demuestre que

  1. $\displaystyle{\left(\liminf_{n\to\infty} A_n\right)^c = \limsup_{n\to\infty} A_n^c.}$

  2. $\displaystyle{\left(\limsup_{n\to\infty} A_n\right)^c = \liminf_{n\to\infty} A_n^c.}$

  3. $\displaystyle{\mathbb{P}\!\left(\liminf_{n\to\infty} A_n\right) = 1 - \mathbb{P}\!\left(\limsup_{n\to\infty} A_n^c\right).}$

  4. $\displaystyle{\mathbb{P}\!\left(\limsup_{n\to\infty} A_n\right) = 1 - \mathbb{P}\!\left(\liminf_{n\to\infty} A_n^c\right).}$

Ejercicio

Sea $\{A_n : n \in \mathbb{N}\}$ una sucesión de eventos. Demuestre que

  1. $\displaystyle{\lim_{n\to\infty} A_n = A \;\Longleftrightarrow\; \lim_{n\to\infty} A_n^c = A^c.}$
  2. $\displaystyle{\lim_{n\to\infty} A_n = A \;\Longleftrightarrow\; \lim_{n\to\infty} \mathbf{1}_{A_n} = \mathbf{1}_A.}$

El símbolo $\mathbf{1}_A$ denota la función indicadora del conjunto $A$. Véase el apéndice al final del texto para la definición y algunas propiedades de esta función.

Ejercicio

Suponga que $\displaystyle{\lim_{n \to \infty} A_n = A}$ y $\displaystyle{\lim_{n \to \infty} B_n = B}$. Diga falso o verdadero. Demuestre en cada caso.

  1. $\displaystyle{\lim_{n \to \infty} \lim_{m \to \infty} (A_n \cap B_m) = A \cap B.}$

  2. $\displaystyle{\lim_{n \to \infty} \lim_{m \to \infty} (A_n \cup B_m) = A \cup B.}$

  3. $\displaystyle{\lim_{n \to \infty} \lim_{m \to \infty} (A_n - B_m) = A - B.}$

  4. $\displaystyle{\lim_{n \to \infty} \lim_{m \to \infty} (A_n \triangle B_m) = A \triangle B.}$

Ejercicio

Medida de probabilidad discreta. Sea $\{x_n : n \in \mathbb{N}\}$ una sucesión de números reales y sea $\{a_n : n \in \mathbb{N}\}$ otra sucesión de números reales no negativos tal que $\displaystyle{\sum_{n=1}^{\infty} a_n = 1}$. Demuestre que la función $\mathbb{P} : \mathcal{B}(\mathbb{R}) \to [0,1]$ definida de la siguiente forma es una medida de probabilidad:

$\displaystyle{\mathbb{P}(A) = \sum_{n=1}^{\infty} a_n \, \mathbf{1}_{\{n : x_n \in A\}}(n).}$

Ejercicio

Sean $\mathbb{P}$ y $\mathbb{Q}$ dos medidas de probabilidad definidas sobre una misma $\sigma$-álgebra. Demuestre que $\alpha \mathbb{P} + (1-\alpha)\mathbb{Q}$ es una medida de probabilidad para cada $\alpha \in [0,1]$.

Ejercicio

Sea $\mathbb{\mathbb{P}}$ una medida de probabilidad. Determine si las siguientes funciones también son medidas de probabilidad:

  1. $1 - \mathbb{P}.$

  2. $\displaystyle{\frac{1 + \mathbb{P}}{2}.}$

  3. $\mathbb{\mathbb{P}}^2.$

  4. $|\mathbb{P}|.$

  5. $4\mathbb{P}(1 - \mathbb{P}).$

  6. $\sqrt{\mathbb{P}}.$

Ejercicio

Determine si las siguientes funciones son medidas de probabilidad.

  1. $\mathbb{P}(\Omega) = 1$ y $\mathbb{P}(A) = 0$ para cualquier otro evento $A$.

  2. $\mathbb{P}(\emptyset) = 0$ y $\mathbb{P}(A) = 1$ para cualquier otro evento $A$.

Ejercicio

Considere el espacio medible $(\mathbb{N}, 2^{\mathbb{N}})$. Demuestre en cada caso que $\mathbb{P}$ es una medida de probabilidad. Para cada $A \in 2^{\mathbb{N}}$ defina:

  1. $\displaystyle{\mathbb{P}(A) = \sum_{n \in A} \frac{2}{3^n}.}$

  2. $\displaystyle{\mathbb{P}(A) = \sum_{n \in A} \frac{1}{2^n}.}$

Ejercicio

Sea $\Omega = \{1, \ldots, n\}$ y considere el espacio medible $(\Omega, 2^{\Omega})$. Investigue en cada caso si $\mathbb{P}$ es una medida de probabilidad. Para cada $A \in 2^{\Omega}$ defina:

  1. $\displaystyle{\mathbb{P}(A) = \sum_{k \in A} \frac{2k}{n(n+1)}.}$

  2. $\displaystyle{\mathbb{P}(A) = \prod_{k \in A} \left(1 - \frac{1}{k}\right).}$
