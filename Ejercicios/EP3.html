Ejercicio

Sea $(X_n)_{n\in\mathbb{N}_{0}}$ una cadena de Markov homogénea con espacio de estados $E$, finito o numerable, y matriz de transición $(p_{xy})_{x,y\in E}$. Demuestre que $$ \displaystyle{ i\to j \quad \Longleftrightarrow \quad f_{ij}>0. } $$

Ejercicio

Sea $(X_n)_{n\in\mathbb{N}_{0}}$ una cadena de Markov homogénea con espacio de estados $E$, finito o numerable, y matriz de transición $(p_{xy})_{x,y\in E}$. Use la identidad: $$ \displaystyle{ p_{ij}(n)=\sum_{k=1}^{n} f_{ij}(k)p_{jj}(n-k), \qquad n\geq 1, } $$ para demostrar que, para cada $n\geq 1$, $$ \displaystyle{ p_{ij}(n)\leq \mathbb{P}(\tau_{ij}\leq n \mid X_0=i). } $$

Observe que la igualdad se cumple, en particular, cuando $j$ es un estado absorbente.

Ejercicio

Sea $(X_n)_{n\in\mathbb{N}_{0}}$ una cadena de Markov homogénea con espacio de estados $E$, finito o numerable, y matriz de transición $(p_{xy})_{x,y\in E}$. Demuestre las siguientes igualdades:

  1. $\displaystyle{\mathbb{P}(\tau_{ij}=1\mid X_0=i)=p_{ij}.}$

  2. $\displaystyle{\mathbb{P}(\tau_{ij}=2\mid X_0=i)=\sum_{k\neq j} p_{ik}p_{kj}.}$

  3. Para todo $n\geq 1$, $$ \displaystyle{ \mathbb{P}(\tau_{ij}=n+1\mid X_0=i) = \sum_{k\neq j} p_{ik}\, \mathbb{P}(\tau_{kj}=n\mid X_0=k). } $$
